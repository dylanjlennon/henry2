Slack-native property research for Buncombe County, NC.

This host serves Slack webhooks and a run-trace API — there is no public UI.

  • POST /api/slack/command — slash command handler
  • POST /api/slack/events — Slack events (app mentions)
  • GET /api/runs — recent run list (auth required)
  • GET /api/runs/:id — single run trace (auth required)
  • POST /api/fetchers/:id — isolated fan-out executor (internal)
  • GET /api/artifact/:id — artifact byte proxy (auth required)
