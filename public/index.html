Henry v2

Buncombe County property research — resolves any address or PIN, runs 12 parallel fetchers across public data sources, delivers documents in ~2 minutes.

Web UI

henry-web-inky.vercel.app — search any address, get the full report.

API

  • POST /api/web/search — resolve address + start run, returns runId
  • GET /api/web/status/:id — poll run progress + fetcher results
  • GET /api/web/history — recent runs
  • POST /api/slack/command — Slack slash command handler
  • POST /api/slack/events — Slack app mentions
  • POST /api/fetchers/:id — isolated fetcher executor (internal)
  • GET /api/artifact/:id — artifact byte proxy
  • POST /api/email/inbound — Postmark inbound webhook

Coverage

Buncombe County, NC only. Street addresses and 15-digit PINs both work.
